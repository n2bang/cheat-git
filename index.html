Git Cheat Sheet by Me

Local Repo

# Command Description
1 git init Khởi tạo một Local Repository
2 git branch -M main Cài đặt nhánh chính
3 git config –global user.name "[name]"
git config –global user.email "[email address]" 		Thiết lập tên người dùng / địa chỉ email
4 git status Kiểm tra trạng thái các files của repository
5 git add fileName Thêm fileName vào trạng thái Staged
6 git add *
git add . 		Thêm tất cả tập tin ở trạng thái Working vào trạng thái Staged
7 git commit --m “Message” Commit files nằm trong trạng thái Staged kèm theo thông điệp “Message”
8 git commit -a --m “Message” Commit files đã hoặc chưa nằm trong trạng thái Staged kèm theo thông điệp “Message”
9 git log Xem log các commit trong Repository (tham khảo thêm các param)
10 git rm --f filename Xóa fileName ra khỏi trạng thái Staged và xóa khỏi ổ cứng (file không còn tồn tại)
11 git rm --cached filename Xóa fileName ra khỏi trạng thái Staged (file vẫn còn tồn tại)
12 git checkout -- filename Quay về trạng thái trước của tập tin filename
13 git commit --amend Nhập 1 commit về commit trước đó

Remote Repo

# Command Description
1 git clone repository [directory] Clone Remote Teposity reposity và đặt vào folder [directory]
2 git remote –v Xem thông tin Remote reposity hiện tại
3 git remote show name Xem thông tin name hiện tại
4 git remote add name repository Add Local Repository lên Remote Repository tại repository và đặt tên nó là name
5 git remote rename name1 name2 Đổi tên name1 thành name2
6 git remote remove name Xóa remote name
7 git pull name branch Kiểm tra sự thay đổi trên Remote Repository và merge vào phiên làm việc hiện tại
8 git push name branch Đẩy dữ liệu lên remote repository
9 git config --global alias.st status Tạo alias

Branch

# Command Description
BRANCH
1 git branch branchname Tạo nhánh mới có tên branchname
2 git log --oneline --decorate Xem đang làm việc với nhánh nào
3 git checkout branchname Chuyển làm việc sang nhánh branchname
4 git checkout -b branchname Tạo nhánh branchname và chuyển làm việc sang nhánh branchname
5 git branch Xem danh sách các nhánh, nhánh nào có dấu * là nhánh chúng ta đang làm việc
6 git branch -v Xem commit mới nhất ở mỗi nhánh
7 git branch --merged Xem danh sách các nhánh đã merge
8 git branch --no-merged Xem danh sách các nhánh chưa được merge
9 git branch -d branchname Xóa nhánh branchname và nhánh này đã được merge
10 git branch -D branchname Xóa nhánh branchname và nhánh này chưa được merge
11 git branch -m oldname newname Thực hiện việc đổi tên nhánh từ oldename thành newname
12 git branch -m newname Đổi tên nhánh hiện tại thành newname
TAG
1 git tag tagname Tạo mới một Lightweight Tag có tên là tagname
2 git tag -a tagname -m “Mesage” Tạo mới một Annotated Tag có tên là tagname cùng với thông điệp
3 git tag -a tagname commitID -m “Mesage” Thêm annotated tag có tên tagname vào commit có ID là commitID
4 git tag Xem tất cả các tag hiện có
5 git tag -n Xem tất cả các tag hiện có và phần miêu tả của task
6 git show tagname Xem thông tin tagname
7 git tag –d tagname Xóa tag có tên tagname
8 git tag -a tagname X -m message Thêm annotated tag tagname vào commit có ID là X
9 git checkout –b branchname tagname Tạo một nhánh mới branchname từ tag đã tồn tại tagname
10 git push name tagname Thêm tagname vào remote name
11 git push name --tags Thêm tất cả tag vào remote name
12 git push name -d tagname Xóa tagname tại Remote repository có tên name
REMOTE BRANCH
1 git remote push remote_name branch_name Đẩy nhánh branch_name từ Local Repo lên Remote Repo
2 git remote show remote_name Xem danh sách các nhánh trên Remote Repo
3 git ls-remote remote_name Xem danh sách các nhánh và tag trên Remote Repo
4 git clone -b branch remote_url Clone một Branch chỉ định từ Remote repo
5 git checkout remote_name/branch_name Chuyển đổi làm việc sang nhánh remote
6 git push origin (--delete|-d) branch_name Xóa một Branch từ Remote repo # Git version 1.7.0 or newer
7 git push origin -d branch_name Xóa một Branch từ Remote repo # Git version 1.7.0 or newer
8 git push origin :branch_name Xóa một Branch từ Remote repo # Git versions older than 1.7.0
9 git fetch (--all|-a|origin) Tìm nạp các thay đổi từ một hoặc tất cả các nhánh từ xa

Conflict

# Command Description
COMPARISONS
1 git diff
git difftool		 Staging vs Working (L-R)
2 git diff HEAD Local Repo vs Working (L-R)
3 git diff --staged HEAD
git diff -cached		 Local Repo vs Staging Area (L-R)
4 git diff master origin/master Local Repo vs Remote Repo (L-R)
5 git diff comitID commitID Comit vs Commit
6 git diff tagname vs tagname Tag vs Tag
STASH
1 git stash Lưu toàn bộ nội dung công việc đang làm dở
2 git stash list Xem lại danh sách các lần lưu thay đổi
3 git stash pop 1
git stash apply 1
git stash pop [stash@{1}](old)
git stash apply stash@{1}(old) 		Áp dụng thay đổi từ stash
4 git stash drop 1
git stash drop stash@{1} (old)
git stash clear		 Xóa một hoặc tất cả các thay đổi không cần thiết

ADVANTAGE

# Command Description
MERGE
1 git merge branchname Merge nhánh hiện tại với nhánh branchname
INTERACTIVE REBASE
1 git rebase branch_name Chạy lại lịch sử commit của branch_name trên nhánh hiện tại
2 git rebase -i HEAD~{3} Chạy lại lịch sử 3 commit trên nhánh hiện tại
3 git rebase -i -r --onto [commit-id-dest] [commit-id-source] Chạy lại lịch sử từ commit-id-source đến commit-id-dest trên nhánh gốc
-i: --interactive -r: --reword
REVERT
1 git revert [commit_id] Lệnh git revert có thể coi như lệnh 'undo'
RESET
1 git reset HEAD filename Xóa fileName ra khỏi trạng thái Staged (file vẫn còn tồn tại)
2 git reset --hard [commit_id] Xóa toàn bộ các commit trước đó và đưa branch về trạng thái của commit có commit_id đã chọn
MOVE
1 git mv (--force|-f) fileName1 fileName2 Đổi tên fileName1 thành fileName2
(--force|-f): cho phép đổi tên để sửa chữ hoa chữ thường trên các hệ thống tệp không phân biệt chữ hoa chữ thường
git config --global core.ignorecase false

.gitignore, .sh or .bat file

# Command Description
Ignore file
1 .gitignore Chứa những file thay đổi nhưng không muốn push mỗi lần commit
2 .gitkeep Tạo file bên trong thư mục muốn xuất bản lên git
SH or BAT file
1 commit.sh 
git add .
git commit -m "$1"
git push
 Thực thi khối lệnh xuất bản code lên git
Linux
sh commit.sh 'add commit sh file'

Windows
replace "$1" -> %1 lưu thành commit.bat
.\commit.bat 'add commit bat file'
